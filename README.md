# NYC Taxi | Proyecto Big Data

Proyecto desarrollado para la materia de Big Data, enfocado en el análisis de movilidad urbana utilizando el dataset **NYC Taxi Trip Records** y herramientas de procesamiento distribuido con PySpark.

## Objetivo

Analizar patrones de movilidad y comportamiento de viajes de taxis amarillos en Nueva York mediante técnicas de Big Data, exploración de datos y aprendizaje automático.

## Dataset

Dataset utilizado:

- **NYC Taxi Trip Records**
- Fuente oficial:
  https://www.nyc.gov/site/tlc/about/tlc-trip-record-data.page

- Tipo de datos utilizados: Yellow Taxi Trips
- Formato: Parquet
